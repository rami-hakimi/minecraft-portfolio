# Minecraft Portfolio — Rami Al-Hakimi

A Minecraft-themed personal portfolio website built with **Next.js 14**, **TypeScript**, and **React**.

## Features
- 🎮 Minecraft hotbar navigation
- 🌙 Day/night cycle (click the sun/moon)
- ❤️ Health bar & XP bar
- ✨ Floating particles and experience orbs
- 📧 Contact form connected to Formspree
- 📱 Fully responsive
- 🧱 Pixel art Steve-style avatar

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- React 18

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repo → Set Framework to **Next.js** → Deploy

## Run Locally
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)
